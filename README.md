# Tableau-Superstore-Dashboard

##  Project Overview
This project was developed during a **data analytics training program** at a training institution.
As part of the training curriculum, an interactive **Tableau dashboard** was created using the **Sample Superstore dataset** to apply dashboard design, KPIs, parameters, and interactivity concepts learned during training.

##  Training Project Context
* **Training Type:** Data Analytics Training Program
* **Purpose:** Hands-on practical implementation of Tableau concepts
* **Focus Areas:**
  * Dashboard layout and formatting
  * Parameters and calculated fields
  * KPIs and business metrics
  * Filters and dashboard actions
 **Dataset Provided By:** Training Institution (Sample Superstore)

##  Project Objectives
* Design a structured two-panel dashboard
* Create dynamic visualizations using a **Metric parameter**
* Implement KPIs including Profit Margin and Return Rate
* Add interactive filters and filter actions
* Publish the dashboard to **Tableau Public**

##  Tools & Technologies
* Tableau Desktop
* Tableau Public
* Calculated Fields
* Parameters
* Dashboard Actions

## Key Features Implemented
* Two-panel dashboard layout (controls on left, visuals on right)
* Metric parameter (Sales, Profit, # Orders)
* 3 KPI cards with dynamic values
* 6 interactive charts updated based on selected metric
* Top 5 products logic based on selected metric
* Show / Hide filters functionality
* Logo-based navigation to Tableau Public

##  Conclusion
This project demonstrates hands-on experience gained during a data analytics training program, showcasing the ability to design interactive Tableau dashboards, implement KPIs, and apply advanced interactivity features to real-world business data.
